# Roadmap

## Phase 1 — MVP (internal)
- Define metrics and baseline
- Dashboard spec (KPIs + use‑case registry)
- Monthly report agent (summary + risks)
- Governance checklist + policy template

## Phase 2 — Beta (public)
- Data connectors (manual import → automated)
- Alerting agent (quality/risk deviations)
- Use‑case lifecycle: propose → approve → track

## Phase 3 — OSS Product
- Multi‑team benchmarking
- Templates for enablement programs
- Plug‑in system for metrics and agents

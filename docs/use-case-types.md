# Use-Case Types (Draft)

AIPulse supports the following standard use-case categories:

1) **Tool Pilot / Instance Test**
   - Evaluate new tools or model instances.
   - Track cost, quality, adoption, and decision outcome.

2) **Workflow Automation**
   - Repeatable operational tasks with clear time savings.

3) **Knowledge / Search**
   - Research, summaries, competitive intel.

4) **Coding / Dev Productivity**
   - PR reviews, refactors, test generation, code navigation.

5) **Quality & QA**
   - Test generation, regression checks, doc QA.

6) **Customer Support / Ops**
   - Triage, response drafts, playbooks.

7) **Content & Documentation**
   - Technical docs, product content, release notes.

8) **Analytics & Reporting**
   - Insights, dashboards, KPI summaries.

9) **Governance & Risk**
   - Policy checks, compliance workflows, risk flags.

10) **Training & Enablement**
   - Coaching, prompt libraries, internal playbooks.

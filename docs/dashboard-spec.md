# Dashboard Spec (Draft)

## Core KPIs
- Adoption (% active users / week)
- Tasks assisted / week
- Time saved vs. review cost
- Rework rate (% corrected outputs)
- Impact on team KPIs (speed, quality, satisfaction)

## Use-Case Registry
Each use case includes owner, objective, success metric, and status.

## Modules (from role SOW)
- **Tooling & PoCs**: idea → PoC → learning → decision pipeline
- **Security & Governance**: risks, access boundaries, secure alternatives
- **Coaching & Enablement**: 1:1s, pair sessions, visible progress
- **AI Syncs**: agenda, decisions, alignment
- **Stakeholder Coverage**: areas served and expansion status

## Risk & Governance
- Open risks
- Compliance flags
- Required approvals

## Agentic Ops
- Monthly report agent
- Alerting agent
- Quality sampling agent

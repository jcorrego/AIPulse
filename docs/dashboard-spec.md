# Dashboard Spec (Draft)

## Core KPIs
- Adoption (% active users / week)
- Tasks assisted / week
- Time saved vs. review cost
- Retrabajo rate (% corrected outputs)
- Impact on team KPIs (speed, quality, satisfaction)

## Use‑Case Registry
Each use case includes owner, objective, success metric, and status.

## Modules (from role SOW)
- **Tooling & PoCs**: pipeline de ideas → PoC → aprendizaje → decisión
- **Security & Governance**: riesgos, límites de acceso, alternativas seguras
- **Coaching & Enablement**: 1:1, pair sessions, progreso visible
- **AI Syncs**: agenda, decisiones, alineamiento
- **Stakeholder Coverage**: áreas cubiertas y estado de expansión

## Risk & Governance
- Open risks
- Compliance flags
- Required approvals

## Agentic Ops
- Monthly report agent
- Alerting agent
- Quality sampling agent
